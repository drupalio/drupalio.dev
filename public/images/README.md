## Imágenes necesarias

Para que esta implementación funcione correctamente, necesitas añadir las siguientes imágenes:

1. `profile-image.jpg` - Una imagen de perfil cuadrada que se mostrará en un círculo
2. `landscape-bg.jpg` - Una imagen de fondo de paisaje, preferiblemente un desierto o montañas

Puedes descargar imágenes gratuitas de sitios como:
- [Pexels](https://www.pexels.com/)
- [Unsplash](https://unsplash.com/)
- [Pixabay](https://pixabay.com/)

Asegúrate de que las imágenes tengan licencias que permitan su uso en tu proyecto.
